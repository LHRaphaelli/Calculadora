## Calculadora

### Descrição

Este projeto apresenta uma calculadora digital com interface moderna e minimalista, desenvolvida com foco em usabilidade e clareza visual. A aplicação realiza as operações matemáticas básicas como adição, subtração, multiplicação, divisão e porcentagem, além de contar com funções como limpeza total AC, exclusão de dígitos DEL e suporte a números decimais. O design segue uma proposta contemporânea em tema escuro degradê, com botões circulares e destaque para a ação principal, proporcionando uma experiência intuitiva e responsiva.
